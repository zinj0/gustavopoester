# Gustavo Poester

Site portfolio de Gustavo Poester, artista visual brasileiro.

## Estrutura

- `index.html`: página inicial do site.
- `html/`: páginas HTML do site.
- `css/`: estilos.
- `js/`: scripts.
- `data/`: catálogos em JSON/JS.
- `images/`: imagens, vídeos e favicon.

## Uso Local

Abra `index.html` no navegador.

## Publicação

Este projeto pode ser hospedado como site estático no GitHub Pages usando a raiz do repositório.

## Direitos

Todo o conteúdo visual, textos, imagens e vídeos pertencem a Gustavo Poester, salvo indicação em contrário.
